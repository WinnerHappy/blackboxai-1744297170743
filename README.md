# Ferdnando Photos

Welcome to **Ferdnando Photos**, a captivating portfolio website showcasing professional photography services. From stunning nature photography to impactful event images, our mission is to capture and share life's beautiful moments.

## Project Overview

Ferdnando Photos is a visually striking website created using modern web technologies. It features a responsive layout optimized for both desktop and mobile users. The site includes various sections such as a hero introduction, a featured works gallery, and detailed categorization of photography styles including nature, portraits, landscapes, and urban photography.

## Installation

To run Ferdnando Photos locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd ferndando-photos
   ```

2. **Open the project**:
   Open the `index.html` file in your preferred web browser to view the home page. Navigation links will direct you to the gallery and contact pages as well.

## Usage

- Click on the **"Ver Galeria"** button on the homepage to explore the collection of photographs and videos.
- Use the contact page to send inquiries or messages directly to Ferdnando Photos.

## Features

- Responsive design for seamless navigation on both mobile and desktop devices.
- Integration of Tailwind CSS for styling, ensuring a modern aesthetic and customizable experience.
- Interactive elements, including dropdown menus and mobile-friendly navigation.
- A vast gallery of images and videos categorized by photography style.

## Dependencies

The project uses the following external libraries:

- **Tailwind CSS** - for utility-first CSS styling.
- **Google Fonts** - for custom font styling.
- **Font Awesome** - for social media icons and navigation elements.

These libraries are included via CDN links directly in the HTML files.

## Project Structure

The project's structure is organized as follows:

```
/ferndano-photos
 ├── index.html        # Home page
 ├── gallery.html      # Gallery page featuring photography categories
 ├── contact.html      # Contact page for inquiries
 └── categories/       # Directory containing subcategories of photos
```

### HTML Files
- **index.html**: The landing page featuring a hero section and a preview of featured works.
- **gallery.html**: A detailed gallery displaying different photography categories, includes sections for both photos and videos.
- **contact.html**: A contact form for user inquiries along with contact details.

Feel free to explore and contribute to the project!
